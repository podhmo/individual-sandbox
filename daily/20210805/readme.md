## go astmpl marker

marker的なものの実装をどうしようか？
うまく書き換えられるようにしておきたいと言うのと引数を渡したい。
あとは、astからgoのコード生成をするときにスキップしたいパラメーター定義などがある。

- 関数を置き換え対象にすることは許すとして、引数をそのまま使うことはできない
- 一方でemitのタイミングで引数を渡したくなる

構造体を明示的にフィールド名を指定せず初期化するやつって引数不足でいけたっけ？
無理そう。何かしらの値が必要そう。

```go
foo := Foo{"foo", nil}
```
