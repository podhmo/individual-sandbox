### 便利さ

- 1ファイルで複数のファイルの定義を提供
- (対応する形式も自由)
- moduleがある
- functionがある
- 自分自身(self)からの相対位置が取れる
- 変数がある
- コメントが書ける
- formatter,linterがあるはず。。
