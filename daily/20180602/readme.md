## ejected

それぞれ以下のようなscriptsが用意されているっぽい

```
scripts/
├── build.js
├── start.js
└── test.js
```

build.jsはproduction用のbuildで、start.jsはdev用のサーバーを立ち上げる感じ。

### test.js

- defaultでwatchの状態
